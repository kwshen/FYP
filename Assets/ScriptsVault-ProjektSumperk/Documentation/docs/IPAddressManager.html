IPAddressManager

Overview

The IPAddressManager script for Unity allows you to manage and retrieve various IP addresses within your Unity project. This documentation provides an overview of its features, example usage, and important notes for integration.

Features

  • Retrieve and display the local IP address of the device.
  • Retrieve and display the static IP address of a specified network interface.
  • Retrieve and display the global IP address (IPv4 format).
  • Retrieve and display the global IP address (IPv6 format).
  • Get Available IPAddresses
  • Add IPAddress
  • Remove IPAddress

Example Usage


// IPAddressManager:
// Attach the IPAddressManager script to a GameObject.
IPAddressManager.Instance.GetCurrentIPAddress();
IPAddressManager.Instance.GetAvailableIPAddresses();
IPAddressManager.Instance.AddIPAddress(ipAddress);
IPAddressManager.Instance.RemoveIPAddress(ipAddress);

Use IPAddressManager to manage and retrieve various IP addresses for your Unity project.

Notes

The IPAddressManager script simplifies the process of working with IP addresses in Unity. Whether you need to display the local IP address, retrieve a static IP address, or obtain the global IP address, this script provides the flexibility and control you need. Customize the script to match your project's specific requirements and enhance the user experience.
