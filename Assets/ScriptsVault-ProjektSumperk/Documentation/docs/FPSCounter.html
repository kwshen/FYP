FPS Counter

Overview

FPS Counter is a Unity script that displays frame rate and system information on the screen. It provides insights into the performance of your Unity application, including frame rate, memory usage, and device information.

Features

  • Real-time display of frame rate (FPS).
  • Memory information, including reserved, allocated, and Mono memory usage.
  • Device information such as screen resolution, OS, GPU, CPU, and RAM details.
  • Customizable and adjustable for different screen refresh rates.
