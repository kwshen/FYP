SwipeManager

Overview

SwipeManager is a versatile Unity script for detecting swipe gestures on touch devices and mouse input. It allows you to easily detect swipes in different directions and trigger actions based on swipe events.

Features

  • Supports detection of swipe gestures in different directions (Up, Down, Left, Right, etc.).
  • Customizable settings for minimum swipe length and triggering swipes.
  • Integration with Unity's UI TextMeshPro (TMP_Text) to display swipe logs on the screen.

Example Usage


// Attach the SwipeManager script to a GameObject in your Unity scene.

// Subscribe to swipe detection events:
SwipeManager.OnSwipeDetected += HandleSwipe;

void HandleSwipe(Swipe swipeDirection, Vector2 swipeVelocity)
{
    switch (swipeDirection)
    {
        case Swipe.Up:
            LogSwipe("Swipe Up detected!");
            // Perform actions for swipe up
            break;
        case Swipe.Down:
            LogSwipe("Swipe Down detected!");
            // Perform actions for swipe down
            break;
        case Swipe.Left:
            LogSwipe("Swipe Left detected!");
            // Perform actions for swipe left
            break;
        case Swipe.Right:
            LogSwipe("Swipe Right detected!");
            // Perform actions for swipe right
            break;
        case Swipe.UpLeft:
            LogSwipe("Swipe Up-Left detected!");
            // Perform actions for swipe up-left
            break;
        case Swipe.UpRight:
            LogSwipe("Swipe Up-Right detected!");
            // Perform actions for swipe up-right
            break;
        case Swipe.DownLeft:
            LogSwipe("Swipe Down-Left detected!");
            // Perform actions for swipe down-left
            break;
        case Swipe.DownRight:
            LogSwipe("Swipe Down-Right detected!");
            // Perform actions for swipe down-right
            break;
        case Swipe.Tap:
            LogSwipe("Tap detected!");
            // Perform actions for tap
            break;
        case Swipe.None:
            LogSwipe("No swipe detected.");
            // Handle when no swipe is detected
            break;
    }
}

// You can customize the swipe handling logic based on your game's requirements.

Notes

The SwipeManager script simplifies the process of detecting swipe gestures in Unity and provides flexibility in customizing swipe detection parameters. You can use it to enhance user interactions in your games or applications.
