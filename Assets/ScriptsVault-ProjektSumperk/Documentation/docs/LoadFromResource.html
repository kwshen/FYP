LoadFromResource

Overview

The LoadFromResource script is a versatile utility for Unity game developers that simplifies the process of loading various types of resources from the Unity Resources folder. Whether you need to load images, audio, JSON data, text, or video, this script provides a convenient way to access and utilize these resources.

Features

  • Load images, audio, JSON data, text, and video resources effortlessly.
  • Supports resource subfolders for organized asset management.
  • Efficient resource caching to improve performance and prevent redundant loading.
  • Handles resource loading errors and provides error logging.
  • Easy-to-use API for seamless integration into your Unity projects.

Example Usage


// Load an image (Sprite) named "PaintIcon" from the "Images" subfolder (if any).
Sprite imageSprite = LoadFromResource.Load<Sprite>("PaintIcon", "Images");

// Load a JSON file named "mydata.json" from the root folder and deserialize it into a custom data class PersonData.
PersonData personData = LoadFromResource.LoadJSON<PersonData<("mydata");

// Load an audio clip named "gameover" from the "Audio" subfolder (if any).
AudioClip audioClip = LoadFromResource.Load<AudioClip>("gameover","Audio");

// Load the prefab named "ResourcesCube2" from the "Subfolder" folder
GameObject loadedPrefab = LoadFromResource.Load<GameObject<("ResourcesCube2", "Subfolder");

// Load a text asset named "mytext" from the "TextFolder" subfolder (if any).
TextAsset textAsset = LoadFromResource.Load<TextAsset<("mytext", "TextFolder");

// Load a video clip named "myVideo" from the "Videos" subfolder (if any).
VideoClip videoClip = LoadFromResource.Load<VideoClip<("myVideo", "Videos");

// Clear the resource cache.
LoadFromResource.ClearCache();

Customize the resource loading according to your project's needs.

Notes

The LoadFromResource script simplifies the resource loading process in Unity, making it easier to manage and utilize assets in your projects. You can use it to load a wide range of resources, whether they are images, audio, JSON data, text, or video files. Be sure to organize your resources in the Unity Resources folder and specify subfolders if necessary for efficient asset management.
