Slow Motion Manager

Overview

The Slow Motion Manager is a versatile Unity script that allows you to create slow-motion effects in your game. It enables you to control the speed of time, creating dramatic and cinematic gameplay moments.

Features

  • Adjustable slow-motion factor to control the degree of slow motion.
  • Smooth transition between normal and slow motion time scales.
  • Customizable key binding to trigger slow motion.
  • Real-time display of the current status on the UI.

Example Usage


public class ExampleUsage : MonoBehaviour
{

    private void Update()
    {
        // Check if the slow-motion key is pressed
        if (Input.GetKeyDown(slowMotionKey))
        {
            if (!isSlowingDown)
            {
                // Start slowing down time
                isSlowingDown = true;
                currentTransitionTime = 0f;
            }
            else
            {
                // Resume normal time
                isSlowingDown = false;
                currentTransitionTime = transitionDuration;
            }
        }

        // Apply slow motion

        // Resume normal time

    }

    // Update the TMP_Text component with the current status
    private void UpdateStatusText()
    {
        if (statusText != null)
        {
            if (isSlowingDown)
            {
                statusText.text = "Slow Motion Active (Press " + slowMotionKey.ToString() + " to Resume)";
            }
            else
            {
                statusText.text = "Normal Time (Press " + slowMotionKey.ToString() + " for Slow Motion)";
            }
        }
    }
}

Attach this script to empty gameObject and press the Spacebar to trigger a show motion effect.

Notes

The Slow Motion Manager script offers a straightforward way to implement slow-motion effects in your Unity game. It provides flexibility in adjusting the slow-motion factor, transition duration, and key binding for triggering slow motion. Additionally, it allows real-time status updates on the UI text.
