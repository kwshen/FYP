Object Pool

Overview

The Object Pool script is used to manage a pool of GameObjects to improve performance by reusing objects instead of creating and destroying them repeatedly.

Features

  • Efficiently manage a pool of GameObjects.
  • Reuse objects to reduce instantiation overhead.
  • Customize pool size and parent transform.

Example Usage


// Create an Object Pool and configure it in the Inspector.
public ObjectPool objectPool;
public int numberOfObjectsToSpawn = 10;

void Start()
{
    // Spawn objects from the pool.
    for (int i = 0; i < numberOfObjectsToSpawn; i++)
    {
        GameObject obj = objectPool.GetPooledObject();
        if (obj != null)
        {
            obj.transform.position = transform.position;
            obj.transform.rotation = transform.rotation;
            obj.SetActive(true);
        }
    }
}

Notes

The Object Pool script is a common pattern used to improve performance in Unity games by reusing GameObjects instead of creating and destroying them frequently.
