Android Permissions

Overview

Android Permissions with Popup Box is a Unity script that simplifies the process of handling Android permissions in your Unity projects. It provides a user-friendly popup interface to explain why specific permissions are required and then requests those permissions.

Features

  • Supports Android M (6.0) and later.
  • Show permission explanation in a popup.
  • Supports multiple permissions at once.
  • Common permissions included: Camera, Phone State, Location, Microphone, Storage, Contacts.
  • Customizable popup messages for each permission.

Example Usage


// Attach the Android Permissions script to your Unity GameObject.
// Customize the UI elements in the Inspector for different permissions.

// Example usage for requesting storage permission:
Permissions permissions = GetComponent<Permissions>();
permissions.PermissionStorage();

Integrate Android permissions with user-friendly explanations into your Unity projects effortlessly.

AndroidManifest.xml

To use Android Permissions with Popup Box, you need to add the following permissions to your AndroidManifest.xml file: 


                    <uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />
                    <uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" />
                    <uses-permission android:name="android.permission.READ_PHONE_STATE" />
                    <uses-permission android:name="android.permission.CAMERA" />
                    <uses-permission android:name="android.permission.RECORD_AUDIO" />
                    <uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />
                    <uses-permission android:name="android.permission.READ_CONTACTS" />

Notes

Android Permissions with Popup Box simplifies the process of requesting permissions from the user and provides a better user experience by explaining why each permission is required. Make sure to add the necessary permissions to your AndroidManifest.xml file for the script to work correctly.
