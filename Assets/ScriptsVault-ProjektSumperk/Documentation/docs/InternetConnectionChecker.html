Internet Connection Checker

Overview

Internet Connection Checker is a Unity script that allows you to monitor the status of your internet connection. It offers three modes of checking: on button click, timed intervals, and continuous monitoring. You can easily integrate it into your Unity projects to ensure your application responds appropriately to network availability.

Features

  • Three checking modes: On Click, Timed, and Continuous.
  • Customizable check intervals for Timed mode.
  • Notifies you when the connection status changes.
  • Displays the connection status on a TextMeshPro (TMP_Text) component.

Example Usage

Example of using the Internet Connection Checker script:


// Attach the Internet Connection Checker script to your Unity GameObject.
// Choose the desired checking mode and configure settings in the Inspector.

// Example usage for checking the connection on button click:
InternetConnectionChecker connectionChecker = GetComponent<InternetConnectionChecker>();
connectionChecker.CheckConnectionOnClick();

Integrate internet connection monitoring into your Unity projects effortlessly.

Notes

Internet Connection Checker is designed to help you ensure a reliable internet connection in your Unity applications. It provides flexibility with multiple checking modes. When using the Timed mode, you can customize the check interval in the Inspector. Make sure to handle the connection status change notifications appropriately in your scripts.
