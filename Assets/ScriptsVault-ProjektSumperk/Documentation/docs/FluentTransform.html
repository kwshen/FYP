Fluent Transform

Overview

FluentTransform is a versatile Unity script for smoothly transforming GameObjects. It allows fluent chaining of methods for moving, rotating, and scaling GameObjects, with customizable duration, easing, and transformation space (Global or Local).

Features

  • Fluent API for moving (MoveTo), rotating (RotateTo), and scaling (ScaleTo) GameObjects.
  • Customizable duration, easing, and transformation space for each transformation.
  • Unity Events for detecting the completion of transformations.
  • Integration with TransformationSettings ScriptableObject for predefined transformation configurations.

Example Usage: With Scriptable


        public class ExampleUsage : MonoBehaviour
        {
            public FluentTransform transformComponent;
            public TransformationSettings settings;
        
            void Start()
            {
                transformComponent.MoveTo(settings.position, settings.duration, settings.transformSpace, settings.easingType, () => Debug.Log("Movement Complete"))
                                  .RotateTo(settings.rotation, settings.duration, settings.transformSpace, settings.easingType, () => Debug.Log("Rotation Complete"))
                                  .ScaleTo(settings.scale, settings.duration, settings.easingType, () => Debug.Log("Scaling Complete"));
            }
        }

Usage Example: Without Scriptable


        public class ExampleUsage : MonoBehaviour
        {
            void StartTransform()
            {
                gameObject.GetComponent()
                  .MoveTo(new Vector3(5.0f, 0f, -20f), 2.0f, FluentTransform.TransformSpace.Global, FluentTransform.EasingType.EaseInOut, () => Debug.Log("Movement Complete"))
                  .RotateTo(Quaternion.Euler(0f, 90.0f, 0), 2.0f, FluentTransform.TransformSpace.Global, FluentTransform.EasingType.EaseInOut, () => Debug.Log("Rotation Complete"))
                  .ScaleTo(new Vector3(5.0f, 5.0f, 5.0f), 3.0f, FluentTransform.EasingType.EaseInOut, () => Debug.Log("Scaling Complete"));
            }
        }

Unity Events

You can use Unity Events to handle various events related to transformations:

  • onMovementComplete: Triggered when a movement transformation is completed.
  • onRotationComplete: Triggered when a rotation transformation is completed.
  • onScalingComplete: Triggered when a scaling transformation is completed.

You can assign methods or functions to these Unity Events to perform custom actions when transformations are finished.

Notes

The script uses coroutines for smooth and efficient transformations over time. Easing functions enhance the visual quality of transformations. TransformSpace allows specifying whether transformations are applied relative to the world or the GameObjectâ€™s local space.
