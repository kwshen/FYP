Camera Shake

Overview

The Camera Shake script is a powerful tool for adding dynamic camera shake effects to your Unity games. It allows you to create immersive and cinematic experiences by simulating camera vibrations, impacts, or other intense events.

Features

  • Adjustable parameters for controlling shake duration, intensity, frequency, and smoothness.
  • Option to randomize shake intensity for more natural effects.
  • Compatible with any camera in your Unity project.
  • Easy integration with your game's events or inputs to trigger camera shakes.

Example Usage


public class CameraShakeExample : MonoBehaviour
{
    private CameraShake cameraShake;

    private void Start()
    {
        cameraShake = GetComponent();
    }

    private void Update()
    {
        if (Input.GetKeyDown(KeyCode.Space))
        {
            cameraShake.Shake();
        }
    }
}

Attach this script to empty gameObject and press the Spacebar to trigger a camera shake effect.

Notes

The Camera Shake script provides a straightforward way to enhance the realism and excitement of your Unity game. Feel free to adjust the script's parameters to achieve various types of camera shake effects, such as explosions, earthquakes, or character impacts.
